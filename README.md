# Portfolio Website

## Overview
This is my personal portfolio website developed using React and Vite.  
It showcases my skills, projects, certifications, and contact information.

---

## Technologies Used
- React.js
- Vite
- HTML
- CSS
- JavaScript
- Git & GitHub
- Vercel

---

## Features
- Responsive portfolio design
- About Me section
- Skills section
- Projects showcase
- Contact section
- Social media links

---

## Development Process
1. Created React project using Vite.
2. Designed responsive user interface.
3. Added portfolio sections and styling.
4. Added images and project details.
5. Used GitHub for version control.

---

## Deployment Process
1. Uploaded project to GitHub.
2. Connected GitHub repository with Vercel.
3. Configured Vite deployment settings.
4. Deployed portfolio website successfully.

---

## Challenges Faced
- Git remote configuration errors
- Image importing issues
- GitHub push conflicts
- Deployment setup issues

---

## Solutions Implemented
- Fixed Git remote URL

- Used correct Git commands
- Configured Vercel correctly for Vite project
